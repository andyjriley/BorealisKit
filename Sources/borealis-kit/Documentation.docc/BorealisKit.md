# ``BorealisKit``

A comprehensive SwiftUI component library following UCM (User Component Model) patterns from the Borealis Component Library.

## Overview

BorealisKit provides a complete set of reusable UI components and design system elements for building consistent, accessible iOS applications. The library includes components for buttons, cards, text, inputs, chips, progress indicators, and more, all following the Borealis design system.
